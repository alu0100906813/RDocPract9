    class Plato

    Clase Plato¶ ↑

    Contiene un conjunto de alimentos, junto con su cantidad en gramos. Además le podemos añadir un nombre.

    Attributes

    alimentos[R]

    Lista que contiene todos los alimentos del plato

    cantidad[R]

    Array que contiene la cantidad de cada alimento en gramos

    carboh_totales[R]

    Contiene los carbohidratos totales de todos los platos

    gramos[R]

    Contienen los gramos totales de todos los platos

    lipid_totales[R]

    Contiene los lípidos totales de todos los platos

    nombre[R]

def get_huella()
    # Primero empezamos con la energía
    vct = get_vct()
    if(vct<670.0)
        a=1
    else
        if(vct<830.0)
            a=2
        else
            a=3
        end
    end
    hc = get_huella_carbono(0)
    if(hc<800.0)
        b=1
    else
        if(vct<1200.0)
            b=2
        else
            b=3
        end
    end
    c = (a+b)/2.0
    c
end

def get_huella_carbono(iter)
    return 0.0 if iter == @alimentos.size()
    return @alimentos.at(iter)["value"].get_gei() + get_huella_carbono(iter+1)
end

    Añade un alimento al plato

    # File lib/plato.rb, line 41
def push(alimento, cantidad)
    @alimentos.push(alimento)
    @cantidad.push(cantidad) # En gramos
    div = cantidad/100.0 # Vemos cuantos gramos totales tenemos de cantidad respecto a la porción
    @gramos += cantidad
    @prot_totales += (alimento.get_proteinas()*div) # Añadimos las proteinas totales
    @carboh_totales += (alimento.get_carbohidratos()*div) # Añadimos los carbohidratos totales
    @lipid_totales += (alimento.get_lipidos()*div) # Añadimos los lípidos totales
end
