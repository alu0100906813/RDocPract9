    Alimentos¶ ↑

    • Asignatura de: Lengajes y paradigmas de programación

    • Alumno: Francisco Javier Arocas Herrera (Alu0100906813)

    Instalación¶ ↑

    Ejecuta esto en el terminal para poder ejecutar la nueva gema.

    gem 'alimentos'

    Luego ejecuta:

    $ bundle

    O también puedes instalarlo como

    $ gem install alimentos

    Uso¶ ↑

    Para utilizar, solo ejecuta el comando

    $ rake

    Dentro del directorio de alimentos, te saldrán las respectivas pruebas
